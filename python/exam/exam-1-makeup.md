# 阶段考核1-基础知识实践-补考

### 目标
编写一个程序，实现用户输入一段语句，输出语句的反向。如：
```sh
请输入语句：most valuble person
most valuble person反向后是nosrep elbulav tsom
```

### 要求
1. 把相应功能定义为一个函数，程序调用函数处理
2. 代码必须有相应注释
3. 把自己的姓名拼音当作测试结果截图
4. 程序保存为exam-1.py

### 提示信息
- for i in 字符串 : 使用for循环可以循环字符串的每一个元素

### 验证程序
运行程序，输入相应信息，检查输出内容是否符合要求

注意以下几点：
- 测试程序结果是否和例子中一致
- 使用自己姓名的拼音进行测试，结果是否正确
